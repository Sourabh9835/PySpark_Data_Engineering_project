# Delta Lake and PySpark Project

Welcome to **PySpark Project** repository!
This project demonstrate a comprehensive PySpark concept using Delta Lake Table and analytics solution which generate actionable insights. Designed as a portfolio project, it highlights industry best practices in Data Engineering and Analytics.

---

## Project Requirement 

### Building Delta Table using PySpark (Data Engineering)

#### Objective
-**Data Source**: Import data from source system provided as a csv file.
-**Data Quality**: cleanse and resolve data quality issue prior to analysis.
-**scope**: focus on latest dataset only.
-**Documentation**: Provides clear documentation of the data model to support both business stakeholder and analytics teams.

---

### Analytics and Reporting (Data Analytics)

### Objective
-**About description quantity**
_**Monthly sales trend**
_**Weekly sales trend**

This insights empowers the stakeholder with key business metrices, enabling strategic decision-making.
